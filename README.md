# Modelagem-de-Banco-de-Dados
Este repositório contém os projetos realizados na disciplina de Modelagem de Banco de Dados. Os trabalhos abordam desde o design de bancos de dados até a implementação de consultas SQL, com foco em boas práticas de modelagem e normalização de dados.

## Projetos

- **[Projeto Conceitual](#)**: Foi realizado o projeto conceitual, utilizando o BrModelo. E a representação gráfica foi feita através do DER, Diagrama Entidade-Relacionamento, criada por Peter Chen para facilitar a visualização dos resultados de uma modelagem.
- **[Projeto Lógico](#)**: Foi feita a documentação do esquema relacional do projeto conceitual, onde fizemos a transformação de conceitual para lógico.
- **[Projeto Físico](#)**: Foi criado o banco de dados físico com base no modelo conceitual, desenvolvido com tabelas, chaves primárias e chaves estrangeiras. E também, teve o dicionário de dados.
